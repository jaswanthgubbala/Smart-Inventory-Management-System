# Smart Inventory Management System

## 📌 Project Description

This project is a **Smart Inventory Management System** developed using Python and MySQL.
It helps in managing product details, tracking stock levels, and handling sales efficiently.

---

## 🚀 Features

* Add new products
* View all products
* Update product quantity
* Delete products
* Process product sales
* Low stock alert
* Dashboard showing total products and stock

---

## 🛠️ Technologies Used

* Python
* MySQL
* VS Code

---

## 🗄️ Database

* Database Name: `inventory`
* Table Name: `products`
* Fields:

  * id (Primary Key)
  * name
  * price
  * quantity

---

## ▶️ How to Run

1. Create database in MySQL
2. Run the SQL table creation code
3. Install mysql connector:

   ```
   pip install mysql-connector-python
   ```
4. Run the Python program
5. Follow menu options

---

## 📊 Output

* Console-based menu system
* User can perform all inventory operations

---

## 🎯 Purpose

This project is useful for small businesses to manage stock and avoid product shortages.
